# Travel Planner

Single-file travel itinerary planner for desktop and mobile browsers.

## Use

Open the GitHub Pages URL, then use the app menu to load or export an itinerary JSON file.

The app code and itinerary data are separate:

- `index.html`: app UI and logic
- `itinerary.json`: trip data that you keep in Google Drive, iCloud, or local storage

## Data

Exported JSON preserves:

- days and spots
- route mode and calculated segment data
- spot notes
- timeline fields: arrival time, stay minutes, and leave time
